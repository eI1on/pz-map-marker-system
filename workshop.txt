title=[B41] Map Marker System
description=This mod introduces an advanced in-game interface for creating and managing custom map markers on both the world map and mini-map. This mod is particularly useful for [b]server administrators[/b] and [b]players organizing events[/b], providing versatile tools for marking locations, areas, and points of interest with ease.
description=
description=[h1]How to Access the Map Marker System[/h1]
description=[b]Singleplayer:[/b]
description=[list]
description=[*]Open the Debug Menu and select the "Map Marker System" button.
description=[*]Use the World Object Context Menu to find the "Map Marker System" option.
description=[/list]
description=
description=[b]Multiplayer:[/b]
description=[list]
description=[*]Open the Admin Menu and select the "Map Marker System" button.
description=[*]Alternatively, access it through the Debug Menu.
description=[/list]
description=
description=[h1]Features[/h1]
description=The mod comes packed with numerous features to help you customize and manage markers effectively:
description=[list]
description=[*][b]Comprehensive Marker Management:[/b] Add, edit, delete, and organize multiple markers with an intuitive user interface
description=[*][b]Import/Export System:[/b] Save your marker configurations to JSON files and share them between servers or restore from backups
description=[*][b]Current Location Picking:[/b] One-click button to automatically set marker coordinates to your current player position
description=[*][b]Admin Teleportation:[/b] Instantly teleport to any marker location for quick navigation (admin feature)
description=[*][b]Dynamic UI:[/b] Interface elements automatically arrange based on marker type for optimal user experience
description=[*][b]Real-time Updates:[/b] All changes sync immediately across the server for multiplayer consistency
description=[/list]
description=
description=[h1]Marker Types & Capabilities[/h1]
description=The mod includes three distinct types of markers, each with unique capabilities:
description=
description=[h2]1. Texture Marker[/h2]
description=Displays custom textures (icons, images, or symbols) on the map.
description=[b]Features include:[/b]
description=[list]
description=[*][b]Custom Textures:[/b] Use any image file path, including built-in examples like [code]media/ui/quest1.png[/code]
description=[*][b]Smart Scaling:[/b] Choose between fixed scale or zoom-based scaling that adapts to map zoom level
description=[*][b]Zoom Control:[/b] Set maximum zoom level for marker visibility
description=[*][b]Size Customization:[/b] Adjust texture scale for perfect visibility
description=[/list]
description=
description=[h2]2. Rectangle Marker[/h2]
description=Creates customizable rectangular shapes with full color and text control.
description=[b]Features include:[/b]
description=[list]
description=[*][b]Advanced Text Options:[/b] Enable/disable marker names, different fonts, customize colors and scaling
description=[*][b]Precise Positioning:[/b] Set exact center coordinates with width and height controls
description=[*][b]Full Color Control:[/b] Customize both marker and text colors with RGB values
description=[*][b]Smart Scaling:[/b] Fixed scale or zoom-based scaling with maximum zoom level control
description=[/list]
description=
description=[h2]3. Area Marker[/h2]
description=Defines large areas by selecting two corner points for zone marking.
description=[b]Features include:[/b]
description=[list]
description=[*][b]Corner-based Definition:[/b] Set Northwest and Southeast corners for precise area coverage
description=[*][b]Customizable Outlines:[/b] Full RGB color control for area boundaries
description=[*][b]Text Integration:[/b] Same advanced text options as rectangle markers
description=[*][b]Zoom Control:[/b] Maximum zoom level setting for optimal visibility
description=[/list]
description=
description=[h1]Advanced Features[/h1]
description=[list]
description=[*][b]Font Variety:[/b] 22 different font options including Small, Medium, Large, Handwritten, Dialogue, and more
description=[*][b]Zoom-based Visibility:[/b] Control when markers appear based on map zoom level for clutter-free navigation
description=[*][b]Server Logging:[/b] Complete admin action logging with player details, Steam IDs, and role information
description=[*][b]Multiplayer Sync:[/b] Real-time marker updates across all connected players
description=[*][b]Persistent Storage:[/b] Markers automatically save and persist between server restarts
description=[/list]
description=
description=[h1]Perfect For[/h1]
description=[list]
description=[*][b]Server Administrators:[/b] Mark important locations, safe zones, and event areas
description=[*][b]Event Organizers:[/b] Create temporary markers for community events and activities
description=[*][b]Roleplay Servers:[/b] Mark faction territories, trading posts, and points of interest
description=[*][b]New Player Guides:[/b] Mark safe houses, water sources, and essential locations
description=[*][b]Community Projects:[/b] Coordinate building projects and resource gathering areas
description=[/list]
description=
description=[h1]Technical Notes[/h1]
description=[list]
description=[*][i]Area markers cannot be filled with color due to game engine limitations (outline only)[/i]
description=[*][i]Area marker names may not render on the mini-map in some cases[/i]
description=[*][i]Requires ElyonLib and StarlitLibrary dependencies[/i]
description=[*][i]Compatible with Build 41+[/i]
description=[/list]
