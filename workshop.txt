title=[B41] Map Marker System
description=[h1]Map Marker System[/h1]
description=This mod introduces an advanced in-game interface for creating and managing custom map markers on both the world map and mini-map. This mod is particularly useful for [b]server administrators[/b] and [b]players organizing events[/b], providing versatile tools for marking locations, areas, and points of interest with ease.
description=
description=[h1]How to Access the Map Marker System[/h1]
description=[b]Singleplayer:[/b]
description=[list]
description=[*] Open the Debug Menu and select the "Map Marker System" button.
description=[*] Use the World Object Context Menu to find the "Map Marker System" option.
description=[/list]
description=
description=[b]Multiplayer:[/b]
description=[list]
description=[*] Open the Admin Menu and select the "Map Marker System" button.
description=[*] Alternatively, access it through the Debug Menu.
description=[/list]
description=
description=[h1]Features[/h1]
description=The mod comes packed with numerous features to help you customize and manage markers effectively:
description=[list]
description=[*] Add, edit, and delete custom markers with unique names, locations, and textures.
description=[*] Enable or disable markers to toggle visibility.
description=[*] Customize marker scale: fixed scaling or zoom-based scaling and maximum zoom level for marker visibility.
description=[*] Teleport to the location of a marker with ease (admin feature).
description=[*] Use a detailed user interface to manage multiple markers intuitively. The UI elements arrange dynamically based on marker type.
description=[*] To add a new marker a different modal will open when clicking the Add Marker button in the main menu
description=[/list]
description=
description=[h1]Marker Types[/h1]
description=The mod includes three distinct types of markers, each with unique capabilities:
description=
description=[h2]1. Texture Marker[/h2]
description=Displays a custom texture (e.g., icons or images) on the map.
description=[b]Options include:[/b]
description=[list]
description=[*][b]Fixed Scale:[/b] Prevents the marker from resizing when zooming.
description=[*][b]Maximum Zoom:[/b] Controls the marker's visibility based on the zoom level.
description=[*][b]Size:[/b] Adjust the texture scale.
description=[*][b]Custom Textures:[/b] Allows the use of unique textures via file paths. For example, you can use the image in this mod's files: [code] /media/ui/quest1.png [/code]
description=[/list]
description=
description=[h2]2. Rectangle Marker[/h2]
description=A customizable rectangle.
description=[b]Options include:[/b]
description=[list]
description=[*][b]Name customizations:[/b] Being able to enable/disable the marker name, different fonts, color and scale.
description=[*][b]Coordinates:[/b] Define the rectangle's center position on the map.
description=[*][b]Size:[/b] Adjust the rectangle's width, height and scale.
description=[*][b]Color:[/b] Fully customizable with RGB values.
description=[*][i]Includes [b]Fixed Scale[/b] and [b]Maximum Zoom[/b] options.[/i]
description=[/list]
description=
description=[h2]3. Area Marker[/h2]
description=Used for delimiting larger areas by selecting two corner points.
description=[b]Options include:[/b]
description=[list]
description=[*][b]Corner Coordinates:[/b] Define the area's two corners, top corner (North-West) and bottom corner (South-East).
description=[*][b]Outline Color:[/b] Fully customizable with RGB values.
description=[*][i]Includes [b]Name customizations[/b] and [b]Maximum Zoom[/b] options.[/i]
description=[/list]
description=
description=[h1]Known issues[/h1]
description=[list]
description=[*][i]The area marker cannot be filled with color due to a game engine limitations.[/i]
description=[*][i]The marker name for the area type doesn't render on the mini-map, I don't know why.[/i]
description=[/list]
