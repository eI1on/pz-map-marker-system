description=[h1]Map Marker System[/h1]
description= This mod provides a powerful in-game interface for managing custom markers on the in-game world and mini maps. Useful for server wide events.
description=
description= How to Open the Map Marker System:
description=
description= Singleplayer:
description= In the debug menu, the "Map Marker System" button.
description= In the world object context menu, the "Map Marker System" option.
description=
description= Multiplayer:
description= In the admin menu, the "Map Marker System" button.
description=
description=[h1]Features[/h1]
description=
description=[list]
description=[*] Add new markers with custom names, locations, and textures.
description=[*] Edit and delete custom markers markers.
description=[*] Teleport to the location of a marker.
description=[*] Enable/disable markers.
description=[*] Set the scale of the marker texture.
description=[/list]
description=
description= Please note: This is a developer branch of the mod, which means it may be unstable and contain bugs.
description= The mod's functionality may be subject to change in future updates.
description=
