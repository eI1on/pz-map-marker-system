title=Map Marker System [DEV Branch]
description=[h1]Map Marker System[/h1]
description=This mod introduces an in-game interface for managing custom markers on both the world map and mini-map, making it ideal for organizing server-wide events.
description=
description=[h1]How to Access the Map Marker System[/h1]
description=[b]Singleplayer:[/b]
description=[list]
description=[*] Open the Debug Menu and select the "Map Marker System" button.
description=[*] Use the World Object Context Menu to find the "Map Marker System" option.
description=[/list]
description=
description=[b]Multiplayer:[/b]
description=[list]
description=[*] Open the Admin Menu and select the "Map Marker System" button.
description=[*] Alternatively, access it through the Debug Menu.
description=[/list]
description=
description=[h1]Features[/h1]
description=[list]
description=[*] Add, edit and delete new markers with custom names, locations, and textures.
description=[*] Enable/disable markers.
description=[*] Set the scale of the marker texture.
description=[*] Teleport to the location of a marker.
description=[/list]
description=
description=[i][b]Note:[/b] This is the developer branch of the mod, which may include bugs or experimental features.[/i]
description=[i]The mod's functionality may be subject to change in future updates.[/i]
description=
